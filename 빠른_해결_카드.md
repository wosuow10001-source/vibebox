# ⚡ 빠른 해결 카드 - Netlify 401 에러

## 🎯 3단계로 해결 (5분)

### 1️⃣ Netlify 환경 변수 추가

https://app.netlify.com → 1vibebox 사이트 선택

```
Site configuration → Environment variables → Add a variable
```

**추가할 변수 3개:**

| Key | Value | Scopes |
|-----|-------|--------|
| `DEV_LOGIN` | `true` | All scopes |
| `DEV_EVAL` | `true` | All scopes |
| `NEXT_PUBLIC_BASE_URL` | `https://1vibebox.netlify.app` | All scopes |

### 2️⃣ 재배포

```
Deploys 탭 → Trigger deploy → Deploy site
```

2-3분 대기 → "Site is live" 확인

### 3️⃣ 테스트

https://1vibebox.netlify.app/admin 접속

✅ 401 에러 사라짐
✅ 관리자 페이지 로드됨

## 🔍 빠른 체크

## 🐛 여전히 안 되면?

1. **환경 변수 오타 확인**
   - DEV_LOGIN (대소문자 정확히)
   - 값: true (따옴표 없이)

2. **재배포 확인**
   - 환경 변수 추가 후 재배포 했는지
   - 빌드가 성공했는지

3. **브라우저 캐시**
   - Ctrl + Shift + R (하드 리프레시)
   - 시크릿 모드에서 테스트
