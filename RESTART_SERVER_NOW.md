# 서버 재시작 필요

## 수정 완료 ✅

`SiteRenderer.tsx`의 tags 처리 로직을 수정했습니다.

### 수정 내용
- tags가 배열이 아닐 경우 처리
- tags가 문자열 배열일 경우 처리
- tags가 객체 배열 {tag: {name, slug}} 형식일 경우 처리

## 서버 재시작 방법

### 1. 현재 서버 중지
터미널에서 `Ctrl + C` 누르기

### 2. 서버 재시작
```bash
npm run dev
```

## 예상 결과

서버가 정상적으로 시작되고 다음 URL에서 페이지가 로드되어야 합니다:
- http://localhost:3000

## 확인 사항

✅ 500 에러가 사라짐
✅ 메인 페이지가 로드됨
✅ "사탄" 영상 카드가 표시됨
✅ 콘솔에 에러 없음

## 문제가 계속되면

서버 터미널의 에러 메시지를 확인하고 전체 에러 로그를 공유해주세요.
