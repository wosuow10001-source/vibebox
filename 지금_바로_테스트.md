# 🎬 지금 바로 영상 업로드 테스트

## ✅ 서버 실행 완료!
```
http://localhost:3000 - 서버 실행 중
```

## 📋 테스트 순서

### 1단계: 직접 업로드 페이지 접속
```
http://localhost:3000/admin/upload-direct
```
브라우저에서 위 주소를 열어주세요.

### 2단계: 영상 파일 선택
- 파일: `korea_hiphop_ai_musicvedio_satan1.mp4` (42MB)
- "영상 파일 선택" 버튼 클릭
- 파일 선택하면 자동으로 업로드 시작

### 3단계: 업로드 진행 확인
- 진행률 바 표시됨
- "청크 1/9 업로드 완료..." 메시지
- 약 1-2분 소요 (네트워크 속도에 따라)

### 4단계: 메인 페이지 확인
```
http://localhost:3000
```
- 업로드 완료 후 자동으로 메인 페이지로 이동
- 영상 썸네일 카드 확인
- 재생 버튼 아이콘 확인

### 5단계: 영상 재생 테스트
- 영상 썸네일 클릭
- 상세 페이지로 이동
- 영상 플레이어에서 재생 버튼 클릭
- 42MB 영상 전체 재생 확인

## 🔍 문제 확인 방법

### 파일 크기 확인
PowerShell에서 실행:
```powershell
Get-ChildItem "public\uploads\asset-*\index.mp4" | Select-Object Name, Length | Sort-Object Length -Descending | Select-Object -First 1
```

예상 결과:
```
Name        Length
----        ------
index.mp4   44040192  ✅ (42MB - 완전한 파일)
```

만약 `10485760` (10MB)이면 여전히 문제가 있는 것입니다.

- **관리자 대시보드**: http://localhost:3000/admin

- **기존 파일 업로드**: http://localhost:3000/admin/upload
- **메인 페이지**: http://localhost:3000

## ⚠️ 주의사항

1. **업로드 중 페이지를 닫지 마세요**
   - 청크 업로드가 중단됩니다
   
2. **진행률이 멈추면**
   - 브라우저 콘솔(F12) 확인
   - 오류 메시지 확인
   
3. **업로드 실패 시**
   - 페이지 새로고침
   - 다시 파일 선택
   
4. **포트 변경 시**
   - 서버가 3001 포트로 시작되면
   - 모든 URL의 3000을 3001로 변경
