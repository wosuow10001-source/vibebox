- 영상이 메인 페이지에 안 보임

## 🔍 지금 당장 확인할 것

### 1. 브라우저 개발자 도구 열기
```
F12 키 누르기
```

### 2. Console 탭 확인
1. Console 탭 클릭
2. 빨간색 에러 메시지 확인
3. 에러 메시지 전체 복사

### 3. Network 탭 확인
1. Network 탭 클릭
2. 영상 업로드 시도
3. presign 요청 찾기
4. upload 요청 찾기
5. 각 요청의 상태 코드 확인 (200, 413, 500 등)
6. Response 탭에서 에러 메시지 확인

### 영상 상세 페이지에서:
- http://localhost:3000/a/sample-content 접속
- 영상 플레이어가 보이나요?
- 재생 버튼을 누르면 영상이 재생되나요?

### 직접 영상 URL:
- http://localhost:3000/uploads/asset-aee476e1-076a-4728-8238-3087ef352f80/index.mp4
- 영상이 직접 재생되나요?
- 404 에러가 나나요?

## 💡 예상 문제들

### 문제 1: 메인 페이지에 영상 카드가 안 보임
→ 브라우저 캐시 문제
→ Ctrl+Shift+R로 하드 리프레시

### 문제 2: 영상 카드는 보이지만 썸네일이 안 보임
→ video 태그 로딩 문제
→ 브라우저 콘솔 에러 확인

### 문제 3: 영상 상세 페이지에서 재생 안 됨
→ 파일 경로 문제
→ 직접 URL 접속해서 확인

### 문제 4: 42MB 업로드가 10MB로 잘림
→ Next.js 바디 크기 제한
→ 서버 터미널에서 에러 확인
